## 🚀 快速启动步骤

### 第 1 步：打开监控
```bash
# Windows 用户：双击
start_monitor.bat

# 或命令行
python "import requests.py"
```

### 第 2 步：等待警报
- 系统每 10 分钟自动检查一次
- 发现关键词会立即推送

### 第 3 步：配置调整（可选）
- 修改关键词
- 添加新闻源
- 调整检查间隔

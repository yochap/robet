# 🚀 闻泰科技事件实时监控系统

## 📋 功能说明

本系统通过爬取多个新闻源，实时监控与闻泰科技相关的重要事件，并在发现关键词时自动推送警报。

## 🔔 警报触发机制

### 利好警报 (🟢 BUY ALERT)
当新闻标题包含以下关键词时触发：
```
✓ 中国全面豁免 Nexperia 芯片出口管制
✓ 中荷达成正式和解
✓ 荷兰撤销对 Nexperia 管理层的限制
✓ 荷兰法院恢复中方 CEO 职权
✓ Nexperia 恢复大规模出货
✓ 闻泰科技公布季度营收反弹
✓ 闻泰科技宣布欧洲扩产
✓ 中国放宽半导体出口
✓ 中欧启动半导体产业合作
✓ 德国高层访华并签署产业合作协议
```

### 利空警报 (🔴 SELL ALERT)
当新闻标题包含以下关键词时触发：
```
✓ 中方重新收紧出口
✓ 荷兰法院裁定 Nexperia 
✓ 资产冻结
✓ 德国或欧盟统一外资审查强化
✓ 欧洲车企供应停产
```

## 🧪 快速测试

### 方法 1: 使用测试模式（推荐）

1. 打开 `import requests.py` 文件
2. 找到第 61 行，改为：
   ```python
   TEST_MODE = True
   ```
3. 运行脚本：
   ```bash
   python "import requests.py"
   ```
4. 程序会立即发送 3 条测试警报，然后退出
5. 检查你的手机/邮箱是否收到通知

### 方法 2: 手动测试警报

```bash
python test_alerts.py
```

## ⚙️ 配置说明

### 修改监控间隔
在 `import requests.py` 第 59 行：
```python
INTERVAL = 600  # 改为其他数值（单位：秒）
# 例如：
INTERVAL = 300   # 5 分钟
INTERVAL = 60    # 1 分钟
```

### 添加新的关键词
在 `import requests.py` 中找到对应的关键词列表：

**添加利好关键词：**
```python
BULL_KEYWORDS = [
    # ... 已有的关键词 ...
    "你想要监控的新关键词",  # 添加这一行
]
```

**添加利空关键词：**
```python
BEAR_KEYWORDS = [
    # ... 已有的关键词 ...
    "你想要监控的新关键词",  # 添加这一行
]
```

### 修改 PushPlus SendKey
如果想要更改推送密钥，编辑第 60 行：
```python
PUSH_PLUS_SENDKEY = "你的新SendKey"
```

## 🔧 故障排除

### 问题 1: 没有收到推送
- ✓ 检查 SendKey 是否正确
- ✓ 确认网络连接正常
- ✓ 先运行测试模式验证推送功能
- ✓ 查看 PushPlus 官网 https://www.pushplus.plus/

### 问题 2: 抓取失败（403、401错误）
- ✓ 这些是正常的（某些网站禁止爬虫）
- ✓ 系统会自动尝试其他新闻源
- ✓ 可以添加更多新闻源 URL

### 问题 3: 没有触发警报
- ✓ 确保关键词正确拼写
- ✓ 关键词匹配**不区分大小写**
- ✓ 运行测试模式确认警报功能正常
